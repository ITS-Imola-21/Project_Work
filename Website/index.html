PROJECT WORK

Project Goal

The project aims to enhance the functionality of home automation for users by leveraging widely used software technologies such as Alexa and integrating IoT-based automations (Internet of Things).
The idea is to create a home automation system that is easier to use and allows users to control their devices quickly and intuitively.

Details

The project for the development of a skill for home assistants aims to create a simple and accessible user interface to interact with home automation devices in one's residence. The skill will be free and developed using current and widely-used software technologies, and it will be accessible through low-cost commercial micro appliances, such as Amazon Echo.
When a user issues a command through the micro appliance, it is sent to the cloud for interpretation. The cloud then sends the response back to the micro appliance, which directs it to the target device. This target device is a laptop connected to the home network, which will be used for programming and interacting with a microcontroller.
The chosen microcontroller for this project is a classic 5V Arduino, known for its stability, robustness, and ability to interact with a wide range of devices. The use of a low-voltage microcontroller would result in less versatility and robustness, as well as additional technical challenges.
Additionally, a diagram outlining all communications, protocols, timing sequences, and diagnostic and logging mechanisms will be designed to ensure the security of the home network. The hypothesis of a man-in-the-middle attack will also be explored, where the DNS is modified to redirect traffic to a rogue server.
The project will be documented on this website, providing information about the project, statistics, and links to the code on the GitHub repository. This website will serve as a reliable source of information for anyone interested in learning more about the project.
