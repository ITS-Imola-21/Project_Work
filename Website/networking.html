Networking

Network architecture and security

The cloud development team and the microcontroller development team collaborated in designing the network architecture of the project, with a particular focus on creating graphical diagrams for the entire network infrastructure.
An analysis of the network traffic between the Echo Dot and Amazon servers was conducted using the packet analyzer software Wireshark. This allowed for the generation of graphs and statistics that were useful for telemetry of communications between the Echo device and Amazon servers.
The team took care of producing and installing an SSL certificate on the web server used for hosting the project's website, ensuring a secure connection through the encrypted HTTPS protocol.

Graphs

The group used Wireshark software as a protocol analyzer to monitor network traffic between the Echo Dot device and Amazon Alexa servers.
The program was set up to capture packets from the two IP addresses, and two 5-minute recordings were made - one without issuing any commands to the Echo Dot and the other with five voice inputs given. After recording the data, they used Wireshark's Conversations menu to view the data for each recording. This tool was useful for assessing the difference in bandwidth and the number of packets sent based on interactions with the device, providing valuable information for telemetry of communications between the Echo Dot and Amazon Alexa servers.
