Website

Website for controlling Echo Dot data

The website aims to showcase the work done by various teams and provide detailed information about the activities and achievements. The website serves as the platform that hosts information about the project and the functionalities of the created system.
On the website, it is possible to access the history of sent commands, including precise dates and times, to have a comprehensive view of the project and evaluate its success. A web page has been developed that contains a form for registering each user of Amazon Alexa (Echo Dot) for the project. The link to access this page is https://pw2023.itsimola.it/form/registration.php.

Details

To support the management of data collected from the Echo Dot device, a MySQL database has been created. The database consists of 6 tables and one view.
This database has been designed to efficiently handle the information collected from the device, allowing for structured storage. Additionally, an Application Programming Interface (API) has been created to receive telemetry data and insert it into the database tables.
This API has greatly simplified data management, enabling the automation of the data acquisition and storage process. The API acts as an intermediary between the Echo Dot device and the database, allowing the system to collect, process, and analyze the collected data efficiently and accurately.
